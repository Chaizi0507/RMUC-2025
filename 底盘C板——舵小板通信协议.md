底盘C板——舵小板通信协议

MScB通信为CAN 2.0通信接口，波特率1Mbps

#### 指令id

| 指令id | 指令参数        |
| ------ | --------------- |
| 0x1a   | a轮组的矢量速度 |
| 0x1b   | b轮组的矢量速度 |
| 0x1c   | c轮组的矢量速度 |
| 0x1d   | d轮组的矢量速度 |
